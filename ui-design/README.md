## Principes

- Interface minimaliste, claire et professionnelle
- Palette rouge, blanc et gris uniquement
- Cards blanches avec bordures legeres
- Sidebar fixe sur desktop, menu mobile sur petits ecrans
- Transitions courtes de `200ms ease`
- Icones issues de `lucide-react`

## Composants UI

- Button
- Input
- Badge
- Modal
- Card
- Avatar
- Spinner
- EmptyState
- StatCard
- Table
- Pagination
